# SQL Student Database Analysis

This project demonstrates basic SQL skills by creating and analyzing a student database.

## Features
- Create a student table
- Insert sample student data
- Perform data analysis using SQL queries

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregation functions (AVG, COUNT)

## Files
- schema.sql → Creates the table and inserts data  
- queries.sql → Contains analysis queries  

## Example Queries
- Retrieve all students
- Find students with GPA above 3.5
- Calculate average GPA by major
- Count number of students per major

## What I Learned
- Writing SQL queries for real-world data
- Data filtering and aggregation
- Structuring and analyzing datasets
